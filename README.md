# Todo - Консольное приложение для управления задачами

Простое консольное приложение для управления списком задач, написанное на C#.

##  Описание

**Todo** — это простое консольное приложение, позволяющее создавать, просматривать и управлять списком ваших задач . Приложение предоставляет базовый функционал для эффективного планирования.

##  Функциональные возможности

- **Добавление задач**: Создание новых задач с описанием
- **Просмотр списка**: Вывод всех текущих задач
- **Отметка о выполнении**: Возможность пометить задачу как выполненную
- **Удаление задач**: Удаление задач из списка
- **Работа с файлами**: Сохранение списка задач в файл и загрузка из него
